AGENTS
======

Purpose
-------
This file documents the automated agents used in this repository: what they do, their responsibilities, how to operate them in development, and what information maintainers should provide when adding or updating an agent.

What to include for each agent

For all network requests, use tanstack query and place the queries in the lib directory in the root. 

Please add a short description of each small feature addition to the FEATURES.md file

When adding code, include code comments that can help me understand what is going on, I can remove them later.
